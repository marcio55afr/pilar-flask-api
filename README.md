# Pilar API - flask
Aplicação web utilizando Flask com duas funções principais, contagem de vogais e ordenação em listas de palavras. Solução desenvolvida para avaliação técnica na empresa Pilar.

# Windows Guide
## Installation
python setup.py

## How to run
venv\Scripts\activate
cd flaskr
$env:FLASK_APP='app'
flask run

## How to test
venv\Scripts\activate
pytest
